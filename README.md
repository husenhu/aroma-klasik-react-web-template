# ☕ Aroma Klasik - React Web Template

**Aroma Klasik** adalah template website modern untuk Cafe & Restaurant yang dibangun menggunakan **React** dan **Tailwind CSS**. Template ini dirancang dengan estetika klasik namun tetap responsif dan fungsional, cocok untuk bisnis kuliner yang ingin menonjolkan suasana dan menu andalan.

## ✨ Fitur Utama

  * **Hero Section:** Tampilan utama yang memikat dengan CTA (Call to Action) yang jelas.
  * **Highlight Menu:** Section khusus untuk menampilkan menu best-seller dengan harga dan rating.
  * **Facilities & Ambience:** Galeri foto untuk menunjukkan suasana cafe.
  * **Interactive Location:** Integrasi Google Maps dan detail jam operasional yang informatif.
  * **Social Proof:** Section testimoni pelanggan dan Instagram feed grid.
  * **Reservation Form:** Form booking meja yang siap diintegrasikan dengan backend.
  * **Responsive Design:** Optimal di desktop, tablet, maupun smartphone.

## 🛠️ Tech Stack

  * **Frontend:** [React.js](https://reactjs.org/)
  * **Styling:** [Tailwind CSS](https://tailwindcss.com/)
  * **Build Tool:** [Vite](https://vitejs.dev/)
  * **Icons:** Lucide React / FontAwesome
  * **Fonts:** Inter & Playfair Display (via Google Fonts)

## 🚀 Cara Menjalankan Project

### 1\. Clone Repository

```bash
git clone https://github.com/husenhu/aroma-klasik-react-web-template.git
cd aroma-klasik-react-web-template
```

### 2\. Install Dependencies

```bash
npm install
```

### 3\. Jalankan Server Development

```bash
npm run dev
```

Akses di browser pada alamat `http://localhost:3000` (atau port yang muncul di terminal).
